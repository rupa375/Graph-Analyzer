# Graph Algorithms Visualizer (C++)

## 📌 Description
This project is a console-based Graph Algorithms Visualizer implemented in C++. It demonstrates core graph algorithms such as BFS, DFS, Dijkstra’s shortest path, and Prim’s Minimum Spanning Tree using STL data structures.

## 🚀 Features
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra’s Shortest Path Algorithm
- Prim’s Minimum Spanning Tree (MST)
- Menu-driven interactive program
- Efficient graph representation using adjacency list

## 🛠️ Technologies Used
- C++
- STL (vector, queue, priority_queue)
- Graph Data Structures
